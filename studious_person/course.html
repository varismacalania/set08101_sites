BEng(Hons) Cyber Security and Forensics

This is the first course in the UK to recieve full accreditation from the National Security Centre (NCSC).

This course focuses primarily on networking, security and forensics which are all currently the most sought after areas in computing. The first 2 years focus on generic computer studies, these being:

  • Practical Networking 1 + 2
  • Python
  • Javascript
  • Database Systems
  • Operating System studies

3rd and 4th are the more specialist years with a lot more magnification on Network and Computer security with some cyptography thrown in.

  • Data Analytics
  • Network Security and Cryptography
  • OS Forensics
  • Security Testing and Advances Network Forensics
